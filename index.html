Connect Supabase to save your changes permanently

Currently your edits are saved locally in this browser only. To make changes permanent and visible to all visitors, follow these steps:

  1. Go to supabase.com → Create a free account → New Project
  2. In your project go to Table Editor → New Table → Name it portfolio_state
  3. Add 2 columns: id (text, primary key) and data (jsonb)
  4. Go to Settings → API → copy your Project URL and anon public key
  5. Open your index.html → find YOUR_SUPABASE_URL and YOUR_SUPABASE_ANON_KEY → replace with your values → re-upload to Netlify
